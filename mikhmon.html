MIKHMON

MikroTik Hotspot Monitor (Mikhmon) adalah aplikasi berbasis web (MikroTik API PHP class) untuk membantu manajemen Hotspot MikroTik. Khususnya MikroTik yang tidak mendukung User Manager.
Mikhmon bukan radius server, jadi tidak harus selalu aktif dan diaktifkan saat dibutuhkan atau sesuai kebutuhan.
